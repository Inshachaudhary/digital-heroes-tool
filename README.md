 🚀 Digital Heroes Tool

📌 Overview
Digital Heroes Tool is a lightweight, browser-based text utility application built using HTML, CSS, and JavaScript.  
It provides real-time text analysis features with a clean and responsive user interface.

This project was developed as part of a selection/trial evaluation task to demonstrate frontend development skills, DOM manipulation, and UI design thinking.


✨ Key Features
- ⚡ Real-time word and character counter  
- 🧹 One-click text clearing  
- 📋 Copy text to clipboard functionality  
- 🎨 Clean, responsive, and minimal UI  
- 📱 Mobile-friendly design  



🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  


🌐 Live Demo
🔗 Live Project:  
npx plugins add vercel/vercel-plugin


📁 Project Structure
index.html → UI structure  
style.css → Styling and layout  
script.js → Application logic  


⚙️ How It Works
- User enters text in the input area  
- JavaScript listens to input events  
- Word and character count updates in real time  
- Buttons perform actions like clear and copy  
- UI updates dynamically without page reload

This project demonstrates:
- Frontend development fundamentals  
- DOM manipulation using JavaScript  
- Clean UI/UX implementation  
- Ability to build functional real-world mini tools
